About

As you might be aware that we are currently going through a humanitarian crisis i.e. COVID-19 which has changed the face of the earth. It has created a new normal to which we are now abiding to. If one feels like they have the symptoms for COVID-19 they should get themselves checked. But in a country like India where the population is so huge and the number of tests that can be organized are not of the same scale. This is the time where this project comes in handy and fulfils its service. For the people who are unable get an appointment or have to wait some time before they can get themselves checked for COVID-19, they can now get themselves tested by using this project which uses machine learning and analyses the chest X-rays of the individual and detects whether the person is suffering from COVID-19 or not with Acuracy of 92%.This prediction is based on mutiple model outputs. We have used various Deep Learning models such as GooglNet, ResNet etc.
